# Mobile Weather App (University Project)  
모바일 실시간 날씨 애플리케이션 (대학교 프로젝트)

**Project / 프로젝트명:** Real-time Location-based Weather Application  
**Duration / 개발 기간:** May 28, 2024 ~ June 3, 2024  
**Team Members / 팀원:** [@Kr-Jae](https://github.com/Kr-Jae) / [@Junhwipark](https://github.com/Junhwipark)

<br>

## About the Project / 프로젝트 소개

This project is a real-time weather application developed for a university course.  
이 프로젝트는 대학교 수업 과제로 개발된 실시간 날씨 애플리케이션입니다.

It fetches weather data using a Weather API and uses GPS to detect the user’s current location automatically.  
날씨 API를 통해 실시간 데이터를 가져오며, GPS를 이용해 사용자의 현재 위치를 자동으로 감지합니다.

<br>

## Features / 주요 기능

- Uses Weather API to fetch real-time weather data  
  실시간 날씨 데이터를 가져오기 위해 Weather API 사용  
- GPS integration for automatic location detection  
  GPS를 통해 현재 위치 자동 감지  
- Displays current temperature, humidity, and weather conditions  
  현재 기온, 습도 및 날씨 상태 표시  
- Responsive design for mobile devices  
  모바일 기기에 최적화된 반응형 UI 디자인
